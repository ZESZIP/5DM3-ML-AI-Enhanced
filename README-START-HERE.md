# 5D3 AI Enhanced Magic Lantern — Cinema Build

> **Install:** see **[INSTALL-5D3.md](INSTALL-5D3.md)** — or download **`5DM3-AI-Cinema-INSTALL.zip`**

This repo ships a ready-to-flash Canon **5D Mark III (1.2.3)** package with AI Cinema UX:

- **`Movie → Cinema Record`** — Sony-style full-screen recording setup (resolution, aspect, bit depth, FPS, anamorphic)
- Color-coded menus, LiveView **AI-CINEMA** badge, boot confirmation popup
- RAW MLV (14–8 bit), crop modes, FPS to 120 (hardware-limited)

**Verify after install:** Help menu shows **`5DM3.AI.Cinema.2026`**. If you see `2025-06-20`, replace all files from this repo.
