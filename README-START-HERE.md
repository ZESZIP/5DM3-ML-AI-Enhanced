# 5D3 AI Enhanced Magic Lantern — Cinema OS 2026

> **Install:** see **[INSTALL-5D3.md](INSTALL-5D3.md)** — or download **`5DM3-AI-Cinema-INSTALL.zip`**

This repo ships a ready-to-flash Canon **5D Mark III (1.2.3)** package with **Cinema OS 2026**:

- **Delete menu** → colored tab bar, orange **CINE** workspace, bold 2026 UX
- **`Movie → Cinema Record`** — full-screen recorder with **BEAST 4K25** / **BEAST 1080p120** power modes
- LiveView **AI-CINEMA** badge, anamorphic preview, optimized preview scale while recording raw MLV
- 10-bit+ MLV, crop modes, FPS to 120 (hardware-limited on 5D3)

**Verify after install:** Help shows **`CINE.AI.ENHANCED.5D3`**. If you see `2025-06-20`, replace all files from this repo.
